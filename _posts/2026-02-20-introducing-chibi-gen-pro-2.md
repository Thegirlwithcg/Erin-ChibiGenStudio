## What's New in Version 2?

This release is a complete rewrite from the ground up, designed to give you more control, cleaner topology, and faster performance.

- **New Base Mesh System** — Completely redesigned geometry for smoother deformations
- **Auto-Rigging** — One-click rigging with a production-ready armature
- **Customizable Proportions** — Adjust head-to-body ratios, limb lengths, and more

## Clean Topology, Every Time

One of the biggest pain points for character artists is topology. With Chibi Gen Pro 2, every generated mesh features:

1. Quad-dominant geometry
2. Proper edge flow for facial expressions
3. Animation-ready joint loops
4. Optimized polygon count for game engines

> "I used to spend hours on retopology alone. Chibi Gen Pro 2 gives me a clean base mesh in seconds." — Beta Tester

## Getting Started

The add-on works with **Blender 5.0+** and is compatible with both Windows and macOS. Installation is simple: download the `.zip` file and install it via Blender's Preferences panel.
